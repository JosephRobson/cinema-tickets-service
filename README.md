# Cinema Tickets Service

A simple microservice designed to handle the booking of cinema tickets for some imagined ordering service.

# Contents

1. [Spinup Instructions](##Spinup-Instructions)
2. [Architecture](##_Architecture)

## Spinup Instructions
In a terminal from the project root run `npm start` to start the app or alternatively run `npm run dev` to start the app via nodemon which will watch for code changes and automatically restart the app

**N.B.** dev should not be used in deployment as the app crashing is better handled by kubernetes or the deployment environment
